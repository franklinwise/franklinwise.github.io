SSH Shortcuts !(alias || script)

Tue, Aug 12, 2014

ssh alias

script way:

~/.bashrc

alias myserver='ssh -i .ssh/mykey me@myserver.com'

call like this

$ myserver

ssh way:

~/.ssh/config

Host myserver
    HostName myserver.con
    User me
    IdentityFile ~/.ssh/mykey

call like this

$ ssh myserver

ssh way wildcard:

>= OpenSSH 5.6

* Expand %h to variable on command line
     Host *.*
        Hostname %h
        IdentityFile ~/.ssh/mykey
        ... 
     Host *
        Hostname %h.example.org
        ...

call like this

$ ssh someserver

ssh tunnel port forwarding

script way:

~/.bashrc

ssh -N -L 8080:10.0.9.11:8000 me@mydomain.com -i ~/.ssh/mykey &

ssh way:

~/.ssh/config/

Host myname
    HostName mysshgateway.mydomain.com
    IdentityFile ~/.ssh/mykey
    LocalForward 9988 10.0.7.11:80
    User me
